# 06 - Linux and AD Integration Lab

## Status

Planning and research phase.

## Planned Scope

This lab will cover:
- Linux integration with Active Directory
- Kerberos authentication
- SSSD configuration
- centralized Linux authentication
- domain-based SSH access
- cross-platform identity workflows

## Planned Objectives

- integrate Ubuntu Server with Active Directory
- validate domain-authenticated Linux access
- demonstrate centralized identity across platforms
- explore Linux enterprise authentication concepts
- strengthen cross-platform infrastructure understanding
