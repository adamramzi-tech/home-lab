# 03 - Active Directory Lab

## Status

Planning and research phase.

## Planned Scope

This lab will cover:
- Active Directory Domain Services deployment
- domain controller promotion
- AD-integrated DNS
- organizational units
- users and groups
- centralized identity concepts

## Planned Objectives

- deploy a functional Active Directory domain
- understand centralized authentication workflows
- configure AD-integrated DNS
- establish logical identity organization
- prepare infrastructure for domain-joined clients and future integrations
