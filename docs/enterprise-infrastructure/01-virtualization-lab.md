# 01 - Virtualization Lab

## Status

Planning and research phase.

## Planned Scope

This lab will cover:
- VMware Workstation deployment
- virtual networking concepts
- VM lifecycle management
- snapshots and rollback workflows
- virtual hardware allocation
- Windows Server VM preparation
- isolated enterprise lab networking

## Planned Objectives

- establish a virtualization foundation for enterprise infrastructure labs
- evaluate host resource allocation strategies
- configure virtual networking for future Active Directory deployments
- validate VM deployment and connectivity workflows
- create a scalable platform for future Windows infrastructure projects
